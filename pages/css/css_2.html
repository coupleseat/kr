• 상속받을 수 없는 속성의 태그들

    부모로 부터 자식이 상속받을 수 있는 속성에 폰트가 있다.

    글자 색깔, 폰트 사이즈, 서체명, 줄간격, 두꺼운 정도...?

    예외적으로는 a 태그는 색상만 상속받지 못한다.

    h1 ~ h6 태그는 font-size 를 상속받지 못한다.

    a { color: inherit; }

    h1 { font-size: inherit; }

    h1 { font-size: 100%; }

    100%; 는 inherit; 과 동일하다.

    폼 태그 추가

    폼 컨트롤 요소 모두 부모로 부터 폰트 속성을 상속받지 못한다.

    input, select, textarea, button { font-family: inherit; font-size: inherit; color: inherit; }
