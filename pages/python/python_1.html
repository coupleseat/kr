• 내부, 외부 여백

    padding 테두리 안쪽 내부 여백.

    margin 외부 여백.

    margin: 50px; // 상하좌우의 여백이 동일.

    margin: 50px 35px 20px 0px; // 각각 다른 경우 top부터 시계방향으로 회전해서 한칸씩 띄워쓰기.

    margin: 50px 20px; // 수직과 수평이 동일한 경우 수직을 먼저 작성함.
