• ★ 단축키 사용자 추가하기

    1) 파일 > 기본설정 > 바로가기키 선택
    2) 찾으려는 단어 입력
    3) 원하는 단어를 더블클릭하여 단축키 입력

    **** 추가
    부모태그로 감싸기 : wrap with Abbreviation => Ctrl + Shift + A
    문서 들여쓰기 정렬 : Beautify => Ctrl + Alt + B

  • 단축키 
    셀렉션 위/아래로 복사 : shift+alt+down 또는 shift+alt+up
셀렉션 위/아래로 이동 :  alt+up 또는 alt+down

한 줄(Line)  주석 토글하기 Ctrl + /  (블록이 아닌 한줄 주석이 여러 줄에 적용됨, 여러 줄에도 사용이 가능)
블록 영역만 주석  Shift + Alt + A

블록 선택 된 문자와 같은 문자 선택(바로 뒤 하나씩만 선택됨, 여러개를 선택하고 싶으면 반복함) : Ctrl + D

현재 줄 선택하기  Ctrl + i

선택 라인으로 이동하기 : Ctrl + G

모두 접기 : Ctrl + K 후 Ctrl + 0
모두 열기 : Ctrl + K 후 Ctrl + J
현재 영역만 접기 : Ctrl + Shift + [ 
현재 영역만 열기 : Ctrl + Shift + ]
